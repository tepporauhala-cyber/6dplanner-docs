System settings change the behavior of the application.

## General

The following settings can be quickly toggled on/off:

- **Globe** - Global terrain model and background maps. When off, hides terrain and background maps.
- **Project plane** - Virtual project plane, if import is set to use "Plane" mode
- **Anti-aliasing** - Edge anti-aliasing
- **Shadows** - Shadows from sunlight
- **HUD** - Information display for rendering speed, such as fps
- **Save battery** - Display is not updated when no changes occur in the 3D view
- **Ambient occlusion** - Ambient occlusion effect that simulates light bouncing off surfaces
- **Inspection tool** - Special tool for inspecting model objects
- **Bounding volume** - Shows the volumes of the selected model data

## Settings

The following settings can be fine-tuned:

- **Background color** - Adjust the background color visible when the globe is turned off
