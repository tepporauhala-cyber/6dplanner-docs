The data tree tool is used to manage all project data sources, organize them into folders, manage access rights, and control how data appears in 3D views.

## Toolbar

- **Add data** - Opens a dialog with tabs for different data types. Select the data type, choose data type-specific import settings and press the Add button.
- **Add folder** - Adds a new folder under the currently selected folder.
- **Update data** - Upload a new version of a model file. This creates a new version that appears in the data tree under the model hierarchy.
- **Download** - Download the selected data file.
- **Data settings** - Adjust settings for the selected data source.
- **Rename** - Rename the data source with a new name.
- **Hide** - Make the data source visible / hide it.
- **Zoom to data** - Move the current view to fit the data source.
- **Re-import** - Re-import the selected data source.
- **Delete** - Delete the selected data.
