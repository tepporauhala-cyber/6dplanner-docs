The Topic Report tool is used for reporting and summarizing all project topics into an easily readable table.

## Functionality

- Browse topics using the table with a scrollbar
- Header columns show the topic summary
- If topics have workflows, each workflow has a separate table
- **Export** - Export the topic report to Excel for further use

## Tips

Use the topic report for reporting project status to end clients. The topic report shows every project topic, their status and workflow steps. This is generated automatically and saves your time.
