The Feedback Report is used to combine all feedback into one table to facilitate feedback analysis.

## Functionality

- Scroll through feedback responses in the table
- Export the feedback table as an Excel file for further analysis
- Emoji responses are stored as numbers in the Excel file in order of the emoji icon (default 0 = very happy and 4 = very unhappy)

## Tips

- Perform feedback statistics in Excel by calculating the mean, median, etc. of feedback results
- Analyze feedback with AI by downloading the feedback report to Excel and asking questions to AI
