The Timeline tool is used to change the current time of the project, and can be used for example for shadow analysis.

## Functionality

- Set the exact time with the date and time picker
- Adjust the active time with forward / backward buttons
- Move time with the mouse on the timeline slider
- Play timeline animation with the Play button
- Adjust the timeline's active time scale: hour / day / week / month / year

## Tips

Perform sun shadow analysis with the timeline:
1. Enable shadows in system settings
2. Set the active start time precisely in the time picker
3. Set the time scale to hours
4. Move the timeline slider from left to right to simulate sunlight shadows at different times
5. You can also play the shadow analysis animation with the Play button
