The Snapshot tool is used to summarize the project status into a single-page status report with multiple analytical views.

## Functionality

- **Time range selector** - Select the time range for which the status report is collected
- **Topics Pie and Table** - Shows a summary of topics as a pie chart and table
- **Velocity** - Analysis of how quickly topics progress or are closed
- **Active topics** - Chart of topic progress history
- **Project completion** - Shows how complete the project is based on topic statuses

## Tips

Use this automatic snapshot dashboard to communicate project status to stakeholders.
