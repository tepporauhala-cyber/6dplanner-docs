The Topics tool is a powerful collaboration tool for managing various project topics such as tasks, issues, risks, requirements, comments, notes.

## Topic Editor

### Add and Edit

- Add a new topic using the Add Topic button in the right toolbar
- Open an existing topic by clicking the Topic icon in the 3D view

### Edit Panel

**Basic Information:**
- **Name** - Topic name, displayed in the 3D view above the topic icon
- **Description** - Topic description as multi-line text
- **Type** - Notice / Comment / Idea / Task / Issue / Risk / Requirement
- **Priority** - Low / Normal / High / Critical
- **Status** - Kanban-style status for the topic
- **Tags** - Add your own tags to topics

**Additional Information:**
- **Viewpoints** - Topics can have multiple viewpoints
- **Objects** - List of objects related to the topic

**Workflows:**
Workflows are predefined processes with steps, each step having its own responsible persons and checklists.

**Actions:**
- Viewpoint - Save viewpoints to the topic
- Location - Set the topic's location in 3D
- Link to objects - Select and link objects to the topic
- Save / Cancel
