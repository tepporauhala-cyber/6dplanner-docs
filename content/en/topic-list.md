The Topic List tool is used to manage large numbers of topics easily with sorting and filtering tools.

## Functionality

- **Sort** - Click a column header to sort the content
- **Filter** - Click the filter icon in the column header to define a filter rule
- **Quick filter** - Click a cell value to filter only those values
- **Select** - Select topics using the checkbox on the left side of the table
- **Delete** - Delete selected topics with the Delete button
- **New** - Create new topics

**Note:** If there are many topics, they are listed on multiple pages. You can switch between pages from the page tool in the bottom right corner.

## Tips

- Click your name in the Person field to quickly list topics assigned to you
