The Viewpoints tool is a powerful presentation tool that replaces separate presentation tools like PowerPoint. The Viewpoints tool is used to save predefined views.

## Functionality

- Click a viewpoint to activate it
- Click the Update icon to update the viewpoint
- Click the Delete icon to delete the viewpoint
- Double-click the viewpoint name to edit it

Viewpoints save and load the following settings:

- Camera view
- Timeline time
- Data source visibility and clipping
- Rendering settings such as shadows

## Tips

- Create interactive presentations with viewpoints, like a PowerPoint slideshow
- Share presentations by sharing the project link with viewers
