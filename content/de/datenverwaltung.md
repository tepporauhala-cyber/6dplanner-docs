Das Datenverwaltungs-Werkzeug wird zur Verwaltung aller Projektdatenquellen verwendet, um sie in Ordner zu organisieren, Zugriffsrechte zu verwalten und zu steuern, wie Daten in 3D-Ansichten erscheinen.

## Symbolleiste

- **Daten hinzufügen** - Öffnet einen Dialog mit Registerkarten für verschiedene Datentypen
- **Ordner hinzufügen** - Fügt einen neuen Ordner unter dem aktuell ausgewählten Ordner hinzu
- **Daten aktualisieren** - Laden Sie eine neue Version einer Modelldatei hoch
- **Herunterladen** - Laden Sie die ausgewählte Datendatei herunter
- **Dateneinstellungen** - Einstellungen der ausgewählten Datenquelle anpassen
- **Umbenennen** - Datenquelle mit neuem Namen umbenennen
- **Ausblenden** - Datenquelle sichtbar machen / ausblenden
- **Auf Daten zoomen** - Aktuelle Ansicht auf die Datenquelle verschieben
- **Neu importieren** - Ausgewählte Datenquelle neu importieren
- **Löschen** - Ausgewählte Daten löschen
