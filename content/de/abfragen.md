Das Abfrage-Werkzeug ist eine leistungsstarke integrierte Datenbanksuche, die das schnelle Finden von BIM-Objekten aus großen Projektdaten ermöglicht.

## Funktionalität

- **Neu** - Neue Abfrageregel definieren
- **Speichern** - Bestehende Abfrageregel speichern
- **Ausführen** - Ausgewählte Abfrageregel ausführen

### Regeleditor

- **AND / OR / XOR**-Regeln für mehrere Bedingungen verwenden
- **"Ordner"**-Regel zum Einschränken auf bestimmte Ordner
- **"Attribut"**-Regel zum Suchen nach Attributwerten
- **"Datentyp"**-Regel zum Einschränken auf bestimmte Datentypen

### Ergebnisstil

- **Ergebnis** - Darstellungsstile für Ergebnisse
- **Andere** - Darstellungsstile für andere Daten
- **Farbe** - Objektfarbe
- **Transparenz** - Objekttransparenz
- **Sichtbarkeit** - Ergebnisse erzwungen sichtbar / ausgeblendet

## Tipps

Verknüpfen Sie Abfrageergebnisse mit 4D Gantt-Zeitplanaufgaben für Planung auf Objektebene. So finden Aufgaben immer Objekte basierend auf Abfrageregeln, auch wenn Modelle aktualisiert werden.
