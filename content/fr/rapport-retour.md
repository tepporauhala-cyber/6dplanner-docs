Compile tous les retours dans un tableau unique.

## Fonctionnalité
- Parcourir les réponses dans le tableau
- Exporter en fichier Excel
- Réponses emoji stockées comme chiffres (0 = très satisfait, 4 = très insatisfait)

## Conseils
- Calculer les statistiques dans Excel
- Analyser avec l'IA
