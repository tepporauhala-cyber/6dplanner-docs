Outil de collaboration pour gérer les sujets du projet.

## Éditeur de sujets

### Ajout et modification
- Ajoutez avec le bouton Ajouter un sujet
- Ouvrez un existant via l'icône dans la vue 3D

### Panneau d'édition
**Informations de base :**
- **Nom** / **Description** / **Type** (Avis / Commentaire / Idée / Tâche / Problème / Risque / Exigence)
- **Priorité** (Basse / Normale / Haute / Critique)
- **Statut** / **Tags**

**Informations supplémentaires :**
- **Points de vue** / **Objets**

**Workflows :**
Processus prédéfinis avec étapes et responsables.

**Actions :**
- Point de vue, Emplacement, Lier aux objets, Sauvegarder / Annuler
