L'outil arbre de données gère toutes les sources de données du projet.

## Barre d'outils
- **Ajouter des données** - Dialogue avec onglets pour différents types
- **Ajouter un dossier** - Nouveau dossier sous le dossier sélectionné
- **Mettre à jour** - Télécharger une nouvelle version
- **Télécharger** - Télécharger le fichier sélectionné
- **Paramètres** - Ajuster les paramètres de la source
- **Renommer** / **Masquer** / **Zoomer** / **Réimporter** / **Supprimer**
