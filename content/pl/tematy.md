Narzędzie tematów to potężne narzędzie współpracy do zarządzania tematami projektowymi.

## Edytor tematów

### Dodawanie i edycja
- Dodaj nowy temat przyciskiem Dodaj temat
- Otwórz istniejący klikając ikonę w widoku 3D

### Panel edycji
**Informacje podstawowe:**
- **Nazwa** / **Opis** / **Typ** (Uwaga / Komentarz / Pomysł / Zadanie / Problem / Ryzyko / Wymaganie)
- **Priorytet** (Niski / Normalny / Wysoki / Krytyczny)
- **Status** / **Tagi**

**Informacje dodatkowe:**
- **Punkty widokowe** / **Obiekty**

**Przepływy pracy:**
Predefiniowane procesy z krokami i odpowiedzialnymi.

**Akcje:**
- Punkt widokowy, Lokalizacja, Powiąż z obiektami, Zapisz / Anuluj
