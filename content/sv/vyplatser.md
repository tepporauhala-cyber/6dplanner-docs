Vyplatsverktyget är ett kraftfullt presentationsverktyg som ersätter PowerPoint.

## Funktionalitet
- Klicka på en vyplats för att aktivera den
- Klicka på uppdateringsikonen för att uppdatera
- Klicka på raderingsikonen för att radera
- Dubbelklicka på namnet för att redigera

Vyplatser sparar:
- Kameravy, tidslinjens tid, datakällors synlighet, renderingsinställningar

## Tips
- Skapa interaktiva presentationer
- Dela presentationer via projektlänken
