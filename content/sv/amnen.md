Ämnesverktyget är ett kraftfullt samarbetsverktyg för hantering av projektämnen som uppgifter, problem, risker, krav, kommentarer, anteckningar.

## Ämnesredigerare

### Lägg till och redigera
- Lägg till med knappen Lägg till ämne
- Öppna befintligt ämne genom att klicka på ämnesikonen i 3D-vyn

### Redigeringspanel

**Grundinformation:**
- **Namn** - Visas i 3D-vyn ovanför ikonen
- **Beskrivning** - Flerradig text
- **Typ** - Meddelande / Kommentar / Idé / Uppgift / Problem / Risk / Krav
- **Prioritet** - Låg / Normal / Hög / Kritisk
- **Status** - Kanban-stil status
- **Taggar** - Lägg till egna taggar

**Ytterligare information:**
- **Vyplatser** - Ämnen kan ha flera vyplatser
- **Objekt** - Lista över kopplade objekt

**Arbetsflöden:**
Fördefinierade processer med steg, ansvariga och checklistor.

**Åtgärder:**
- Vyplats, Plats, Länka till objekt, Spara / Avbryt
